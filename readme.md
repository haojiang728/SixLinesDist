# Six Lines PWA 
A plain javascript Single Page App style Progressive Web App (PWA) for Six Lines I Ching divination. 

## Dev Server
Run the local PWA with:
npx serve .

## Auth Secret to call the backend
The Base64‑encoded app secret is stored in:
app.js

Use browser dev tools, and run "btoa('SECRET_CODE')" in the console to generate the hex64 string.

## Deploying to Netlify
This is a static PWA.  
Deploy by dragging the folder to Netlify
